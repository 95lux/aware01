\section{TLV320AIC3204 Register Configuration}
\label{app:codec-regs}

The following tables document the complete I\textsuperscript{2}C register write sequence executed by \texttt{codec\_init} in \texttt{tlv320\_driver.c}.
Registers are listed in write order, grouped by the three phases of the initialisation sequence.
The TLV320AIC3204 uses a paged register map; a write to register 0x00 selects the active page for all subsequent accesses.
Register numbers refer to the active page at the time of the write.

NDAC, MDAC, NADC, and MADC are rate-dependent: the power-enable bit (MSB) is OR-ed with the divider value at runtime.
Table~\ref{tab:codec-rate-dividers} lists the values for each supported sample rate.
