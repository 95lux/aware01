\subsubsection{Electrical Signal Standards}

The Eurorack format defines three signal types that connect modules \cite{TechnicalDetailsA100}:

\begin{description}
	\item[Audio signals] Produced by sound source modules such as oscillators, noise generators or fed externally into the system. 
	Audio signals typically operate at \SI{10}{V_{pp}} ($\pm\qty{5}{V}$).
	
	\item[Control voltages (CV)] Produced by modulation sources such as LFOs and envelope generators. 
	CV signals typically range from \qty{-2.5}{V} to \qty{+2.5}{V} for \ac{LFO}s, and from \qty{0}{V} to \qty{+8}{V} for envelope generators. 
	The V/Oct pitch standard is a specific CV variant, where each \qty{1}{V} increase corresponds to one octave of pitch change.
	
	\item[Gate and trigger signals] Pulse-shaped signals used to start or trigger events. 
	Gate signals are typically \qty{0}{V}/\qty{+5}{V}. Any voltage above \qty{+3}{V} is treated as logic high. 
	Only signals with fast rising edges are suitable -- slowly rising waveforms such as sine or triangle waves are not appropriate as gate or trigger sources.
\end{description}

All three signal types are carried between modules using \SI{3.5}{mm} TS (mono) patch cables, connecting to the \SI{3.5}{mm} jack sockets on each module's front panel.

Eurorack modules are powered via a 10-pin ribbon cable connected to a bus board with 16-pin connectors inside the rack. 
The connectors utilize the standard \qty{2.54}{mm} pitch. 
To prevent accidental reverse power of the power connector it is common to use keyed (polarised) connectors with a mechanical key notch on the module side. 
This ensures that the cable can only be inserted in the correct orientation.

The bus supplies \qty{+12}{V} and \qty{-12}{V} to all modules, with an optional \qty{+5}{V} rail for modules that need it.
The external \qty{5}{V} rail is mostly outdated today though, and modules produce their own \qty{5}{V} rail, if required.

The \qty{-12}{V} line is identified by a red stripe on the ribbon cable, which must always be oriented toward the bottom of the bus connector \cite{TechnicalDetailsA100}.
Many manufacturers mark this on the PCB as a white silkscreen stripe. 

\subsubsection{Eurorack Mechanical Format}


The Eurorack format defines mechanical constraints for all module panels.
Panel height is fixed at \qty{128.5}{mm}, derived from the 3U rack standard ($3 \times \qty{44.45}{mm} = \qty{133.4}{mm}$) minus the clearance required for the mounting rails. 
Panel width is measured in \ac{HP}, where $1\,\text{HP} = \qty{5.08}{mm}$. 
Modules are mounted using M3 screws (DIN\,7985). 
These mechanical dimensions were established by Doepfer with the introduction of the A-100 system and have been adopted as the standard across the modular synthesis industry \cite{ConstructionDetailsA100}.
